# mex_fishing_locations
Geospatial dataset of fish landing sites in Mexico. 

Raw data source:
Ramírez Rodríguez, M., López Ferreira, C., & Hernández Herrera, A. (2004). 
Atlas de localidades pesqueras de Baja California, Baja California Sur y Sonora.
México Instituto Politécnico Nacional, Centro Interdisciplinario de Ciencias
Marinas, Comisión Nacional de Acuacultura y Pesca
